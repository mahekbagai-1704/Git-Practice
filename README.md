## 🔄 Basic Git Workflow

```text
Create Repository
        ↓
git init
        ↓
git status
        ↓
git add .
        ↓
git commit -m "message"
        ↓
git branch
        ↓
git switch <branch-name>
        ↓
git merge <branch-name>
        ↓
git push origin main
```
